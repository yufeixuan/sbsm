## 性能相关名词解释

- QPS每秒查询率(Query Per Second) 

每秒查询率QPS是对一个特定的查询服务器在规定时间内所处理流量多少的衡量标准

- 比特（bit）

意为“位”

- 字节（Byte）

1Byte=8bit
1B/s=8b/s(或1Bps=8bps)

- 1000Mbit/s网卡也称为千兆以太网网卡

所以千兆网卡传输速率125MB/s
